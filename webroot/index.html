frpc web

Open the resident frpc web console.
打开 frpc web 常驻控制台。

Open web / 打开 web Open console / 打开控制台

Default address: http://127.0.0.1:62930
默认地址：http://127.0.0.1:62930

If the page cannot be opened in low memory mode, tap the original module action button first to restore web resident mode.
如果处于最小内存模式导致打不开，请先点击原有模块操作按钮恢复 web 常驻模式。
